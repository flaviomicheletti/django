# Django

Environment Setup:

To set up the environment, follow these steps:

1. Create a virtual environment:

   python3 -m venv .venv

2. Activate the virtual environment:

   . .venv/bin/activate

3. All in one

   python3 -m venv .venv && . .venv/bin/activate

4. Run

   python manage.py runserver
